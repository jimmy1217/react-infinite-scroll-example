# react-infinite-scroll-example

##  安裝
>**```yarn install```**<br/>

## 開發
>**```yarn dev```**<br/>
open http://localhost:3000

## 打包
>**```yarn build```**<br/>

## http-server 部署
>**```yarn start```**<br/>
open http://localhost:3001

## Docker nginx 部署
>**```docker-compose up -d```**<br/>
open http://localhost:3002
